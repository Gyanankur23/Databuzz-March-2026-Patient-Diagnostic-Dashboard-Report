🚀 Patient Diagnostic Report Dashboard (AI-Powered)

📊 Databuzz Power BI Contest – March 2026 Submission

This project is an AI-powered Power BI dashboard designed to analyze patient diagnostic data and generate meaningful healthcare insights for better decision-making.

🔍 Project Overview

The Patient Diagnostic Report Dashboard helps healthcare professionals:

Identify high-risk patients
Analyze clinical parameters
Understand disease patterns
Make data-driven medical decisions

This project combines Power BI + AI tools to accelerate analytics and improve insight generation.

🤖 AI Integration in This Project

This project uniquely leverages AI tools such as:

ChatGPT
Microsoft Copilot
Google Gemini
Claude AI
AI was used for:

✔ DAX formula suggestions
✔ Identifying relationships between medical parameters
✔ KPI and visual recommendations
✔ Faster understanding of healthcare data

📌 Note: AI usage was a mandatory requirement in this challenge.

📊 Dashboard Features
🔹 1. Patient Overview & Risk Segmentation
Categorization of patients into Low, Medium, High risk
Overall patient distribution and trends
🔹 2. Clinical Parameter Analysis

Key health indicators analyzed:

Blood Pressure (BP)
Glucose Levels
Hemoglobin
Thyroid Metrics
BMI (Height & Weight)

📈 Helps identify abnormal patterns and health risks

🔹 3. Correlation-Based Insights (AI Assisted)
Fever vs BP
Thyroid vs Weight
Glucose vs Risk Levels

💡 These insights help understand hidden relationships in patient data

🔹 4. Patient-Level Drill-Through
Individual patient diagnostic view
Medical history & symptoms
Risk classification
Detailed health metrics
🔹 5. KPI & Visual Storytelling
Business-friendly KPIs
Clean and intuitive dashboard layout
Decision-focused visual design
🧠 Key Learnings

✔ AI significantly improves productivity in data analysis
✔ Domain knowledge (healthcare) is critical for meaningful insights
✔ Importance of visual storytelling in dashboards
✔ Enhanced critical thinking & analytical skills
✔ Same dataset → multiple insights → continuous learning

🔥 Key Takeaway

AI is not replacing analysts — it is enhancing how fast and how effectively we think.
